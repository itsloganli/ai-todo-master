# AI-Todo-Master: 3分钟从零构建的智能待办清单
> 本项目由 Bolt.new 协同开发，展示了 AI 辅助编程下的高效交付能力。
>

## 项目亮点 (Highlights)
+ **AI 驱动**：利用 Bolt 提示词工程快速实现复杂逻辑。
+ **响应式设计**：完美适配移动端与 PC 端。
+ **极简美学**：干净的 UI 界面，专注任务管理。

## 技术栈 (Tech Stack)
+ **Core**: React
+ **Styling**: Tailwind CSS
+ **Deployment**: Bolt

## 快速开始 (Quick Start)
```bash
git clone https://github.com/itsloganli/ai-todo-master.git
cd ai-todo-master
npm install
npm run dev
```
